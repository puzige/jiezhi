# 止界 v1.0.0

首个公开版本。

## 功能

- 通过单一开关允许或阻止 Universal Control 指针跨设备移动。
- 启动时读取当前状态，切换后立即应用。
- 常驻菜单栏，可快速切换、显示窗口或退出。
- Universal 二进制，支持 Apple 芯片和 Intel Mac。
- 最低支持 macOS 12.3。

## 安装

1. 下载并打开 `Jiezhi-v1.0.0.dmg`。
2. 将“止界”拖入 Applications 文件夹。
3. 在 Finder 中右键“止界”并选择“打开”。

## 签名说明

v1.0.0 使用 ad-hoc 签名，尚未经过 Apple notarization（公证）。因此首次启动不能直接双击，需要通过右键“打开”确认。应用不会请求管理员权限。

止界依赖 macOS 未公开的 Universal Control 偏好设置；未来系统更新可能影响其可用性。
